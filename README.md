# Luminous Will - Dark Motivation Video Generator

Automated pipeline that creates cinematic dark motivation videos from a topic.

## Setup

Add these secrets in your HF Space settings:
- `ELEVENLABS_API_KEY`
- `PEXELS_API_KEY`
- `PIXABAY_API_KEY` (optional)
- `FREESOUND_API_KEY` (optional)
